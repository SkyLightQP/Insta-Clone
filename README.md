# 인스타그램 클론코딩 백엔드

- [인스터그램 클론코딩 프론트엔드](https://github.com/Sustagram/instagram-clone-front)

## 🐍 Python3 Version
- v3.7.9

## 🚀 시작하기

- venv 사용을 권장

### 📑 의존성 설치

```shell script
pip install -r ./requirements.txt
```

### 💻 개발 서버 시작하기

```shell script
python ./manage.py runserver
```
